Etymology Studio

WooCommerce Website Case Study

Project Overview

Etymology Studio is a fully functional e-commerce website built using WordPress and WooCommerce. It features engraved gifts like candles, bookmarks, and keepsakes.

Inspiration: Women Entrepreneurship

This project was inspired by women entrepreneurs who turn passion into purpose. I wanted the brand to reflect meaning, personalization, and empowerment.

My Role

  • Installed and configured WordPress + WooCommerce
  • Customized the Storefront theme using CSS
  • Designed product pages and layout using block editor

Tools & Technologies

WordPress, WooCommerce, Storefront Theme, Block Editor, CSS, Canva, Google Analytics

Challenges

  • Customizing layout without page builder plugins
  • Styling WooCommerce filters and widgets
  • Implementing A/B testing and responsive layout

Reflection

This project helped me understand how storytelling can be embedded into e-commerce design. I improved my skills in layout control, brand tone, and user experience.
